zkscala by tmillsclare

ZK Scala is a collection of examples and utility classes which help Scala and ZK integration.

ZK Scala include the following: A utility class to help create Scala based ZK components An example of a Scala based ZK component named "simplelabel" Sample of wiring a Scala controller to ZK

Dependencies

Scala framework libraries ZK libraries
